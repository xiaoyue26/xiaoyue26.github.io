java泛型

上界<? extends T>
不能往里存，只能往外取

下界<? super T>
不影响往里存，但往外取只能放在Object对象里

PECS原则
（Producer Extends Consumer Super）原则

  1. 频繁往外读取内容的，适合用上界Extends。
  2. 经常往里插入的，适合用下界Super。
