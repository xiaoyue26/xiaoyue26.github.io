java►设计模式

拷贝工厂

尽量不使用继承，使用聚合。

聚合:雁群与雁
组合:雁与翅膀

除了builder模式，还有一个有用的技巧：拷贝工厂。

cloneable接口和clone方法依赖于整个对象继承树的健壮性，由于设计理念是链式调用父类的clone方法，维护难度过高，出错概率太大。

可行的方案之一是使用拷贝构造函数。

更推荐的方案是使用拷贝工厂，例如下文的:
public static ClassA newInstance(ClassA obj);
